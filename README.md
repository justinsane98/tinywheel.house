### tinywheel.house

# Project setup
- clone repo `git clone https://github.com/justinsane98/tinywheel.house.git`
- Install Jekyll `gem install jekyll`
- start server `jekyll serve` `http://localhost:4000`
- stop server `jekyll stop`
